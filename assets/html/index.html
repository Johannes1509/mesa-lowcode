Getting started

Welcome to "Low-Code MESA" a software platform that enables the simple, initial creation of agent-based simulation models.

This software is aimed at beginners. With the help of the software, simulation models can be created with graphical support and then exported into Python code based on the MESA framework. This can be used to carry out more in-depth implementations. During the creation process, some elements can also be extended with Python code.
